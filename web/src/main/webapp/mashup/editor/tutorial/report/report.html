  2. 双击终点结点，也就是代表柱图那个结点，弹出此结点属性对话框，如图

  3. 选择图中红圈圈着的选项，此时双弹出对说框提示，如图：

    因为不同图的类型，有不同属性值或属性集合， 如果你修改了图的类型，之前图的属性参数自然就不能再用，此对话框是提示你是否真的要改变 图的类型。一般来说，不需要改变图的类型，需是增加一个新的结点来实现新的图，下面将会说明怎么样增加新结点。现在这样做，只是为了说明 此操作。

  4. 最后选择中终点结点，点选工具栏“运行Mashup定义，弹出如下对话框

在HelloWorld基础上新增一个图

  1. 打开编辑器并打开helloworld报表;
  2. 你将看到名为DrawOperator的结点右下角有一个蓝色圆形按钮，如下图所示

  3. 用鼠标点击此蓝色按钮，新结点将会弹出来，并自动连接上DrawOperator,如下图:

  4. 双击此结点，弹出属性对话框，新建结点默认类型为barchart,我们选择piechart,最后确定;
  5. 同样，选中此结点，选择工具栏”运行“按钮，你将看到饼图;
    因为上面操作你已改了原来的图为饼图，所以现在两个都是饼图了，且因为他们的数据相同，所以显示的图形也一样；
